# h-at
Information Extraction App - Spring 2021 NLP 

## Python Requirements
- Python >=3.5 with:
  - nltk (3.5)
  - spacy (3.0.5)
    - https://pypi.org/project/spacy/
    - For the NLP pipeline
  - scikit_learn (0.24.1)
  - wikipedia (1.4.0)
    - https://pypi.org/project/wikipedia/
    - For scraping text from Wikipedia articles for training/verifying/testing
